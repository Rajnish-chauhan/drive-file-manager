# Drive - File Management System

A full-stack cloud storage application designed for secure, streamlined file management. This system utilizes a flat-file architecture focused entirely on direct file uploads and retrieval, ensuring a highly efficient, folder-less user experience.

## Features

* **Secure Authentication:** Enterprise-grade security implemented using Spring Security, featuring both custom login and OAuth2 integration (Google & GitHub).
* **Flat-File Architecture:** Optimized for straightforward data handling; users can directly upload and manage files without the overhead of complex folder hierarchies.
* **Dynamic User Interface:** A responsive, component-driven frontend engineered for seamless interactions and real-time updates.
* **Robust API Design:** Efficient REST APIs handling data persistence and retrieval, optimized with the Java Collections framework.
